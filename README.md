# HTML to Figma — Import Web Pages and Code

Готовый проект из двух частей:

- Figma-плагин с импортом по URL и из локального HTML;
- Render-сервер с Chromium для открытия публичных сайтов.

Самая простая инструкция находится в файле **START-HERE.md**.

## Структура

```text
plugin/       Figma plugin
server/       Node.js + Playwright render service
render.yaml   Render Blueprint
```

## Основные возможности

- импорт публичной страницы по URL;
- импорт локального HTML-проекта;
- editable Figma layers;
- несколько размеров viewport;
- ожидание JavaScript и lazy-loaded секций;
- встраивание доступных изображений;
- блокировка localhost, приватных и зарезервированных IP;
- личный `API_KEY` для тестового сервера.

## Ограничения текущей версии

Это development/MVP-сборка. Она не гарантирует точное воспроизведение страниц с авторизацией, CAPTCHA, WebGL, сложными iframe, закрытыми ресурсами или антибот-защитой. Перед массовой публичной публикацией серверу понадобятся отдельная инфраструктура, мониторинг, лимиты и пользовательская авторизация.
