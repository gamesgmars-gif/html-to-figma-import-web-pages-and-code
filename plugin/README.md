# Figma plugin

Development-плагин с двумя режимами:

- **Import from URL** — через Render-сервер из соседней папки `server`;
- **Local HTML** — выбор HTML и связанных локальных файлов.

Для подключения в Figma Desktop выбери `manifest.json` через **Import plugin from manifest…**.

Полная пошаговая инструкция находится в корневом файле `START-HERE.md`.
